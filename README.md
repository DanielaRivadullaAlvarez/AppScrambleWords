# Scramble Words

Juego de palabras desordenadas en React + TypeScript.  
Mezcla letras, controla intentos y saltos, muestra puntaje en tiempo real y confetti al acertar. UI moderna con Shadcn/ui, hooks avanzados y useReducer para lógica centralizada.

**Cómo jugar:** Adivina la palabra desordenada antes de agotar tus intentos o saltos.

- Shadcn/ui
- useReducer, useEffect
